Introduction

In this project we are going to implement a basic Grocey Store Management System. The purpose of this software is to streamline the operations of a grocery store. This system involves two distinct interfaces: one for managers and one for customers, each interface is tailored to meet their specific needs. The customer interface allows users to browse departments, add items to their cart, and proceed to checkout. The manager interface, on the other hand, provides tools for adding departments and items, setting prices, and saving modifications to a CSV file.

The system involves object-oriented programming principles, including inheritance and polymorphism, to create a modular and maintainable codebase.
